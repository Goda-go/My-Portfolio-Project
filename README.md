## ✨ Key Features

- **Modern & Responsive Design**  
  Works smoothly on Desktop, Tablet, and Mobile devices.

- **Interactive Skills Modal**  
  Clicking on a skill opens a modal with detailed information.

- **Dynamic Sections**  
  Organized sections for:
  - About Me
  - Skills
  - Projects
  - Contact

- **Client-Side Form Validation**  
  JavaScript validation for:
  - Name length
  - Email format
  - Message length

- **Smooth Navigation**  
  Easy header navigation between sections.
